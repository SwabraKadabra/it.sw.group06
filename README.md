# Auftragsverwaltung – Projektarbeit ZbW

## Projektübersicht

Dieses Repository enthält die Projektarbeit **„Auftragsverwaltung“** im Rahmen der Module:

- Datenbanken Advanced (DBA)
- Software Engineering (SWE)
- Programming Advanced II (PGA2)

Ziel ist die Entwicklung einer **Windows-Desktop-Businessapplikation** für die Muster AG zur Verwaltung von:

- Kunden und Adressen
- Artikeln und Artikelgruppen (hierarchisch)
- Aufträgen und Auftragspositionen

Die Projektarbeit wird **agil nach Scrum** durchgeführt und über **Azure DevOps** geplant und dokumentiert.

---

## Technologie-Stack

- **Programmiersprache:** C#
- **UI:** WPF (MVVM)
- **Datenzugriff:** Entity Framework (Code First)
- **Datenbank:** Microsoft SQL Server
- **CI/CD:** Azure DevOps Pipelines
- **Versionsverwaltung:** Git + GitHub
